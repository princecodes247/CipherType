How to use

Simple Typing

let instance = new CipherType("p");
instance .type("Simple text")

Typing multiple lines at a time

Method 1

let instance = new CipherType("p");
instance .type(["Simple text", "Kept simple"])

Method 2

let instance = new CipherType("p");
instance
.type("Simple text")
.type("Kept simple")

Changing text

let instance = new CipherType("p");
instance
.type("Changing text")
.delete(4)
.type("the world...")

Using cursor

let instance = new CipherType("p");
instance
.type("Changing text")
.delete(4)
.type("the world...")

Typing text straight

let instance = new CipherType("p");
instance
.type("Changing text")
.delete(4)
.type("the world...")
